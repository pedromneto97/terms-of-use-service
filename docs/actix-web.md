# Actix-web Adapter

HTTP REST API built with Actix-web, a high-performance, actor-based web framework for Rust. Perfect for traditional REST services and web applications.

## Environment Variables
| Variable   | Description     | Default      |
|------------|----------------|--------------|
| HOST   | HTTP host      | 127.0.0.1      |
| PORT   | HTTP port      | 8080         |

## Quick Setup

### Environment Configuration
```bash
export HOST=0.0.0.0
export PORT=8080
```
