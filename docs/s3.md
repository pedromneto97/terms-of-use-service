# S3 Adapter

AWS S3 storage adapter for securely storing Terms of Use documents and agreements. Provides scalable, durable object storage.

## Environment Variables
| Variable               | Description         | Example         |
|------------------------|--------------------|-----------------|
| AWS_ACCESS_KEY_ID      | AWS access key     | AKIAIOSFODNN7EXAMPLE |
| AWS_SECRET_ACCESS_KEY  | AWS secret key     | wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY |
| AWS_REGION             | AWS region         | us-east-1       |
| S3_BUCKET              | S3 bucket name     | my-terms-bucket |
| AWS_ENDPOINT_URL       | AWS Enpoint        | http://localhost:4566 |
