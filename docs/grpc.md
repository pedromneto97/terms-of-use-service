# Tonic (gRPC) Adapter

High-performance gRPC API using Tonic, a Rust implementation of gRPC. Perfect for microservices and low-latency service-to-service communication.

## Environment Variables
| Variable   | Description     | Default      |
|------------|----------------|--------------|
| GRPC_HOST  | gRPC host      | 127.0.0.1      |
| GRPC_PORT  | gRPC port      | 50051        |

## Quick Setup

### Environment Configuration
```bash
export GRPC_HOST=0.0.0.0
export GRPC_PORT=50051
```
